## 𝐅𝐞𝐚𝐭𝐮𝐫𝐞𝐬

- **Granular Synthesis Core**: High-precision grain scanning and cloud density controls for textural exploration.
- **Celestial Map**: Nebula monochromatic interface.
- **Audio-Reactive Viewport**: A star field that pulsates and vibrates based on real-time synthesis telemetry.
- **Stellar Interaction**: Constellation knobs designed for vertical-drag precision.
- **Celestial Navigation**: Instantly swap between four unique engine profiles: Lyra, Cygnus, Orion and Scorpius.
- **Signal Capture**: Integrated 3-button transport for recording, looping and resetting granular buffers.

## 𝐈𝐧𝐬𝐭𝐚𝐥𝐥𝐚𝐭𝐢𝐨𝐧

### 𝐒𝐭𝐚𝐧𝐝𝐚𝐥𝐨𝐧𝐞
1. Download the latest `Kawn`.
2. Extract & Drag to your `Applications` folder.
3. Open `Kawn` and begin looping.

### 𝐀𝐮𝐝𝐢𝐨 𝐔𝐧𝐢𝐭 (𝐀𝐔) (𝐀𝐛𝐥𝐞𝐭𝐨𝐧 𝟏𝟐)
1. Copy `Kawn.component` to `/Library/Audio/Plug-Ins/Components`.
2. Rescan plugins in Ableton Live 12.
3. Create a MIDI track and load `Kawn` to begin processing

### 𝐕𝐒𝐓𝟑 (𝐀𝐛𝐥𝐞𝐭𝐨𝐧 𝟏𝟐)
1. Copy `Kawn.vst3` to `/Library/Audio/Plug-Ins/VST3`.
2. Rescan plugins in Ableton Live 12.
3. Create a MIDI track and load `Kawn` to begin processing.

## 𝐆𝐫𝐚𝐧𝐮𝐥𝐚𝐫 𝐏𝐫𝐨𝐟𝐢𝐥𝐞𝐬

**Kawn** features four specialized constellation maps that recalibrate the grain engine:

- **LYRA**: Micro-looping profile. Focused on tight grain windows and rapid rhythmic repetitions.
- **CYGNUS**: Mosaic imaging. Wide spatial distribution of grains for shifting textures.
- **ORION**: Atmospheric synthesis. Hazy, overlapping grain clouds designed for drones and pads.
- **SCORPIUS**: Experimental "Strumming." Sharp transients and aggressive grain positioning.

## 𝐂𝐨𝐧𝐭𝐫𝐨𝐥 𝐏𝐚𝐫𝐚𝐦𝐞𝐭𝐞𝐫𝐬

### 𝐆𝐫𝐚𝐧𝐮𝐥𝐚𝐫 𝐄𝐧𝐠𝐢𝐧𝐞
| Parameter | Description |
| :--- | :--- |
| **Activity** | Controls grain density and the internal playback clock speed. |
| **Shape** | Modifies the synthesis envelope (grain window) and transient attack. |
| **Filter** | High-precision master low-pass frequency control. |
| **Mix** | Balance between dry signal and processed granular output. |

### 𝐒𝐩𝐚𝐭𝐢𝐚𝐥 & 𝐌𝐚𝐬𝐭𝐞𝐫𝐲
| Module | Parameter | Function |
| :--- | :--- | :--- |
| **Void** | Reverb / Decay | Dedicated deep-space resonance and tail management. |
| **Space** | Depth / Res | Secondary high-resonance filter and multi-tap delay imaging. |
| **Mastery** | Magnitude | Overall engine intensity and harmonic gain boost. |
| **Mastery** | Event Horizon | Asymmetrical control for grain singularity and saturation thresholds. |
| **Mastery** | Quasar | High-frequency harmonic excitation and "air" mix. |

### 𝐋𝐨𝐨𝐩𝐞𝐫
| Command | State | Result |
| :--- | :--- | :--- |
| **Rec** | Red Glow | Initiates real-time signal recording into the granular buffer. |
| **Play** | Green Glow | Commences playback of the captured celestial loop. |
| **Stop** | Neutral | Terminates audio playback and flushes the active buffer. |
